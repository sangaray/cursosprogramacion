# Programming Basics Glossary

### Informatics (Computer Science)

- **Definition:** A set of scientific knowledge and techniques that make the automatic processing of information possible through computers.
- **Context from Video:** It represents the foundational concept that every programmer should understand regarding automated information handling.

### Computer

- **Definition:** A digital electronic machine equipped with a large capacity memory, capable of solving both logical and mathematical problems using computer programs.
- **Context from Video:** Also referred to as an "ordenador," it is the core physical system that executes operations.

### Hardware

- **Definition:** The physical, material, or tangible components of a computer system.
- **Examples:** Monitors, keyboards, printers, mice, and any internal components that can be physically manipulated.
- **Context from Video:** Defined as the tangible elements of the machine that can be handled directly with your hands.

### Software

- **Definition:** A collection of instructions, data, or computer rules that execute specific tasks inside a computer system. Commonly referred to as programs.
- **Examples:** Antivirus software (e.g., Avast), web browsers (e.g., Firefox), image editors (e.g., Photoshop), and word processors (e.g., Microsoft Word).
- **Context from Video:** The intangible logical layer essential for making the hardware perform meaningful activities.

### Operating System (OS)

- **Definition:** The system software responsible for coordinating, managing, and controlling the overall activities and resources within a computer.
- **Core Responsibilities:**
  - Controlling data input and output operations.
  - Managing storage space allocations for programs and data.
  - Detecting system and component errors.
- **Examples:** macOS, Microsoft Windows, Linux, iOS (for iPhones), and Android (for smartphones).
- **Context from Video:** Acts as the vital bridge linking hardware and software together to ensure the computer functions seamlessly.

# ID, Accessibility and Navigation

## IDentifier

This attribute is used to specify a unique identity for an HTML element. It helps us to create a unique element in our webpage. It function is to generate a unique identity for an element

The ID is used to:

- Apply specific styles with CSS.
- Manipulate elements with JavaScript.
- Create internal links.

## ID VS CLASS

| Characteristic | ID                                  | Class                        |
| :------------- | :---------------------------------- | :--------------------------- |
| Frequency      | Only once per page.                 | Multiple times.              |
| Use in CSS     | Prohibited (due to best practices). | Recommended.                 |
| Use in JS      | For unique and specific elements.   | For groups of elements.      |
| Accessibility  | Vital for forms and links.          | Does not directly influence. |

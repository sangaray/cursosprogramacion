# Forms Third Parts

## MĂˇs Atributos

- `type` = Specifies the type of input field.

  **Values**
  - `date` = A date field. Generates a date picker.
  - `tel` = A phone number field.

- `max` = Specifies the maximum value of the input field.
- `pattern` = Specifies a regular expression that the input field must match.
- `required` = Specifies that the user must fill in the field before submitting the form.
- `placeholder` = Specifies a short hint that describes the expected value of the input field.
- `min` = Specifies the minimum value of the input field.
- `size` = Specifies the width of the input field. It is a number of characters.

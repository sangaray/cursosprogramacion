# Forms Second Part

## Form Elements

- `<input>` = The input element allows the user to send data to the server.

  **Attributes**
  - `type` = The type of the input element.

    **Values**
    - `checkbox` = A checkbox field. It is used to make multiple selection in a group of options.
    - `reset` = A reset button. It is used to empty the form when clicked.

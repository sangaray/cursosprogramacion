# SVG - Vectorial Graphics

## Elements

- `<svg>` = The SVG element is used to embed SVG content in an HTML document and o draw graphics.
- `<rect>` = The rect element is used to draw a rectangle in SVG.
- `<circle>` = The circle element is used to draw a circle in SVG.

## Attributes

- `width` = The width attribute is used to specify the width of the SVG element.
- `height` = The height attribute is used to specify the height of the SVG element.
- `style` = The style attribute is used to specify the style of the SVG element.

## Properties

- `fill` = The fill property is used to specify the color of the SVG element.
- `stroke` = The stroke property is used to specify the color of the border of the SVG element.
- `stroke-width` = The stroke-width property is used to specify the width of the border of the SVG element.
- `x` = The x property is used to specify the x position of the SVG element.
- `y` = The y property is used to specify the y position of the SVG element.
- `r` = The r property is used to specify the radius of the SVG element.
- `rx` = The rx property is used to specify the x-radius of the SVG element.
- `ry` = The ry property is used to specify the y-radius of the SVG element.
- `cx` = The cx property is used to specify the x-center of the SVG element.
- `cy` = The cy property is used to specify the y-center of the SVG element.

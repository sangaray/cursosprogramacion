# Canvas

It is a canvas element that can be used to draw graphics and animations in HTML. In web programming, it is a code element used to draw graphics, animations, and video games directly in the web browser using JavaScript.

## Elemento

- `<canvas>` = The canvas element is used to draw graphics and animations in HTML.

## JavaScript

- `<script>` = The script element is used to embed JavaScript code in an HTML document.

  **Code**
  - `var` = The var keyword is used to declare a variable in JavaScript.
  - `document` = The document object is used to access the HTML document in JavaScript.
  - `getElementById` = The getElementById method is used to get the element with the specified id in JavaScript.
  - `font` = The font property is used to set the font of the text in JavaScript.
  - `fillStyle` = The fillStyle property is used to set the color of the text in JavaScript.
  - `fillText` = The fillText method is used to draw text on the canvas in JavaScript.
  - `beginPath` = The beginPath method is used to start a new path in JavaScript.
  - `arc` = The arc method is used to draw an arc on the canvas in JavaScript.
  - `stoke` = The stroke method is used to draw a line on the canvas in JavaScript.
  - `fill` = The fill method is used to fill a shape on the canvas in JavaScript.

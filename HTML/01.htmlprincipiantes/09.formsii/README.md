# Forms II

## Elements

- `<input>` = Allows users to input data into a form.
  **Attributes**
  - `type` = specifies the function of the input element
    **Values**
    - `button` = A button element. It is used to trigger an action when clicked.
    - `submit` = A submit button. It is used to submit a form when clicked.
    - `color` = A color picker. It is used to select a color.
  - `maxlength` = Specifies the maximum number of characters allowed in the input field`
  - `value` = Specifies the default value of the input field
    **Values**
    - `radio` = A radio button, it is used to make single selection in a group of options
- `<textarea>` = The textarea element represents a multi-line input control, useful for entering text in a long document.
  **Attributes**
  - `rows` = Specifies the number of rows in a text area
  - `cols` = Specifies the number of columns in a text area

- `<select>` = Represents a control that provides a menu of options.
- `<option>` = Specifies an option in a `<select>` element

## Aditiona Attributes

- `placeholder` = It is a short hint that describes the expected value of an input field.

# Basis Concepts Intro

## What is HTML?

Hypertext Markup Language (HTML) is the code that is used to structure and display a web page and its content.

HTML is not a programming language; it is a markup language that defines the structure of your content.

## Where it comes from?

The nuclear physicist Tim Berners-Lee defined the first version of HTML in 1989. It would later evolve to become HTML5.

HTML5 is the latest version of HTML.

In this version, we find new elements, attributes, and behaviors.

It contains a broader set of technologies that allows websites and applications to be more diverse and far-reaching.

## Hypertext

Hypertext is the foundation of the web.

### What is hypertext?

It is nothing more than text that links to other information, such as a link on page X that takes you to page Y.

## Anatomy of an HTML Element

**Ejemplo**
`<p> HTML COURSE Content</p>`

- `<p>` = opening tag
- `HTML COURSE Content` = content
- `</p>` = closing tag

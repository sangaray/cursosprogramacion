# Structural Semantic

## HTML5 Layout

HTML layout provides a way to properly organize web pages.

**Semantic HTML Layout Elements**

- `<header>`= Defines the header of a page or a specific section.
- `<nav>` = Groups navigation links.
- `<section>` = Groups content that shares the same theme within a page.
- `<article>` = Used to identify content that is independent and self-contained.
- `<aside>` = Extra content that enriches the experience but is optional.
- `<footer>` = Represents the footer of a document or a specific section.

**Nowdays DIV**

The div element is used to group content for styling purposes. It is used as a cntainer.

## Elements

- `<DOCTYPE>` = It is used to specify the document type.
- `<html>` = The root of an HTML document.
- `<head>` = Contains information about the document.
- `<title>` = Defines a title for the document. It's show on the browser tab.
- `<meta>` = Contains metadata about the document.
- `<link>` = Establishes relationships between the current document and an external resource.
- `<style>` = Contains style information for the document.
- `<time>` = The time element represents its contents, along with a machine-readable form of those contents in the datetime attribute. The kind of content is limited to various kinds of dates, times, time-zone offsets, and durations, as described below.
- `<script>` = The script element is used to embed executable code in the browser.
- `<figure>` = The figure element represents some flow content, optionally with a caption, that is self-contained (like a complete sentence) and is typically referenced as a single unit from the main flow of the document.
- `<figcaption>` = The figcaption element represents a caption or legend for the rest of the contents of the figcaption element's parent figure element, if any.

**Attributes**

- lang = Indicates the language of the document.
- charset = Specifies the character set used in the document.
- content = Specifies the content of the document.
- rel = Specifies the relationship between the current document and the linked document.
- type = Specifies the media type of the linked document.
- datetime = Specifies a date and time in the format YYYY-MM-DDTHH:MM:SSZ.

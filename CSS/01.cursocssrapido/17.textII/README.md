# Text II

## Properties

- `text-shadow` = The text-shadow property applies one or more shadows to text.

  **Structure** -`text-shadow: h-shadow v-shadow blur-radius color;`

## Fonts

- `font-family` = The font-family property specifies a prioritized list of font families for text.
- `@import` = The @import rule is used to import a font from an external source.
- `font-face` = The font-face rule is used to specify a font face to be used in a document.
- `@font-face` = The @font-face rule is used to specify a font face to be used in a document. @ is used to import a font from an external source.
- `font-indent` = The font-indent property specifies the indentation of the first line of a text block.
- `line-height` = The line-height property specifies the height of a line box.
- `font-decoration` = The font-decoration property specifies the appearance of the text of an element.

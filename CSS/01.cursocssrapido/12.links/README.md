# Links

They are text to navigate to another page or inside the same page or to another resource.

## Properties

- `text-decoration` = The text-decoration property specifies the appearance of the text of an element. It can be used to underline, overline, or strike through the text.

  **Values**
  - `none` = The text has no decoration.

## Pseudo-Classes

- `:link` = The :link pseudo-class is used to style links that have not been visited.
- `:visited` = The :visited pseudo-class is used to style links that have been visited.

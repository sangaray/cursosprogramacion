# Element Types

These are the different types of elements that can be used in HTML.

-`block` = The block element represents a block of content, such as a paragraph, heading, list, or section. It takes up the full width of the container and can contain other elements. They position themself one after in column not in row. They are `h` elements, `div`, `p`, `ul`, `ol`, `li`, `table`, `tr`, `td`, etc.

-`inline` = They are displayed on the same line as other inline elements and can contain other inline elements. They show up next to each other and are not in a new line. They are `span`, `em`, `strong`, `a`, `img`, `br`, `input`, `label` and `select`. The boxes are adjusted to the size of the text or content instead of the size of the browser. In this kind of elements doesn't have a `width` or `height`, but we can use `padding` and `margin`, but it doesn't respect the bock stepping over each other.

### Working with block and inline elements

**Properties:**

- `display` = This is a CSS property that can be used to change the display of an element. It is used to create a block element or an inline element.
  **Values**
  - `block` = This is a block element. It takes up the full width of the container and can contain other elements. It is used to create a block element. Is the default value for block elements.
  - `inline` = This is an inline element. It does not take up the full width of the container and can contain other inline elements. It is used to create an inline element. Is the default value for inline elements.
  - `inline-block` = This is a block element that is displayed inline. It can contain other elements. It is used to create a block element that is displayed inline.

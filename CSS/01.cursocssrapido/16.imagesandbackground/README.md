# Images and The Background

## Images

**Properties**

- `border-radius` = The border-radius property is a shorthand property for the border-top-left-radius, border-top-right-radius, border-bottom-right-radius, and border-bottom-left-radius properties. It allows you to round the corners of an element.
- `box-shadow` = The box-shadow property is a CSS property that adds shadow effects to an element.

  **Structure**

  `box-shadow: h-shadow v-shadow blur spread color;`

- `background-image` = The background-image property sets one or more background images for an element.

  **Structure**
  - `background-image: url("images/GatitosJugando.png");`

- `background-repeat` = The background-repeat property specifies whether an image should be repeated, repeated in a direction, or not repeated at all.

  **Values**
  - `no-repeat` = The background image will not be repeated.
  - `repeat-x` = The background image will be repeated only horizontally.
  - `repeat-y` = The background image will be repeated only vertically.
  - `repeat` = The background image will be repeated in both directions (horizontally and vertically).

- `background-attachment` = The background-attachment property specifies whether a background image is fixed or scrolls with the page.

  **Values**
  - `scroll` = The background image will scroll with the page.
  - `fixed` = The background image will be fixed in place.

**Notes**

If we have an image in a container, we can manipulate separately the container and the image.

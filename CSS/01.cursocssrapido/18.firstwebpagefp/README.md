# First Web Page (Final Practice)

## Properties

- `background-position` = The background-position property sets the initial position of a background image.

  **Values**
  - `center` = The background image is positioned at the center of the background area.
  - `top` = The background image is positioned at the top of the background area.
  - `right` = The background image is positioned at the right of the background area.
  - `bottom` = The background image is positioned at the bottom of the background area.
  - `left` = The background image is positioned at the left of the background area.

- `overflow` = The overflow property specifies whether to clip content, display scrollbars or show a scrollable overflow.

  **Values**
  - `hidden` = The overflow is clipped, and the rest of the content will be invisible.
  - `scroll` = The overflow is clipped, but the scrollbars are shown.
  - `auto` = The overflow is determined by the content.

**Notes**

- It is used to set de margin of the body to 0, this way the elemenets expand to the whole of the browser.

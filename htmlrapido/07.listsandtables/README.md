# Lists and Tables

## Comments

` <!-- -->` Comment in HTML. This is text that is not visible in the browser, used to explain the code.

## Lists

A list is a collection of items. Lists can be ordered or unordered.

### Unordered Lists

- `<ul>` = unordered list. It is a list of items that do not have a specific order.

### Ordered Lists

- `<ol>` = ordered list. It is a list of items that have a specific order.

### List Elements or Items

- `<li>` = list item. It is an item in a list.

## Tables

It shows data in more than one dimension. It is a collection of rows and columns.

### Table Elements

- `<table>` = table. It is a collection of rows and columns and data in one dimension.

**Table Headers**

- `<thead>` = table head. Block of rows describing the title of each column.
- `<th>` = table header. It is a header cell in a table.

**Table Body**

- `<tbody>` = table body. Block of rows containing the data of the table.

**Table Rows**

- `<tr>` = table row. It is a row in a table.

**Table Cells**

- `<td>` = table data. It is a data cell.

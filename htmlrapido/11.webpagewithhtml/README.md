# Web Page

## Additionals

- `<img>` = Image element. It is used to display images on a web page.
  **Attributes**
  - `alt` = The alt attribute specifies an alternate text for the image, if the image for some reason cannot be displayed.
- `<small>` = The small element represents side comments such as small print.
- `<aside>` = The aside element represents a section of content that is related to the main content, but that is not necessarily part of it. It is often used to include sidebars, ads, or other elements that are not essential to the main content.
- `<nav>` = The nav element represents a section of a page that contains navigation links, or a site map.

## External CSS

- `body` = This gives style to the full body of the page. What you use here applies to all the elements of the page.

## Inline CSS

Actually it is not a good practice to use inline CSS. It is better to use external CSS.

- `border` = This gives style to the border of the element.
- `cellpadding` = This gives style to the padding of the cells of the table.
- `bgcolor` = This gives style to the background of an element.

## CSS in the file head

- `<style>` = This gives to a page inside the file of the page grouped in the head of it.
  **Attributes**
  - `type` = The type attribute specifies the media type of the linked resource.
    **Values**
    - `text/css` = The media type is text/css.

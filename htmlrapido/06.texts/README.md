# Texts

## h Elements

They are heading elements that implements six levels of headings. The use should respect the order of the levels. The most used of them are h1 to h3, the last three could be used but it is unusual and not recommended.

```
h1
h2
h3
h4
h5
h6
```

- `<p>` = paragraph. It is a block-level element that represents a paragraph of text.
- `<strong>`= important text. It is a block-level element that represents strong importance, seriousness, or urgency for its contents. It is typically displayed in bold by user agents.
- `<br>` = break line. It is a block-level element that represents a line break.
- `<section>` = define a section of a document. It is used to group related content, by subject.
- `<article>` = define a independent article. It is used to group independent content, by author, something not asociated to the main content.
- `<cite>` = represent the title of a creative work. The cite element represents a reference to a creative work. It must include the title of the work or the name of the author (person, people or organization) or an URL reference, or a reference in abbreviated form as per the conventions used for the addition of citation metadata.
- `<footer>` = define the footer of a page or section.

## Keyword Shortcuts

`shif+alt+downarrow` = Makes a copy of the selected text.

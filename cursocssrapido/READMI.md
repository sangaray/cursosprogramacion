# First Steps

## In file CSS

It means that the styles are inside the file.

### style element

- `<style>` = `The style element is used to define styles for a document in its head.

  **Attributes**
  - `type` = The type attribute specifies the media type of the linked resource.

    **Values**
    - `"text/css"` = The linked resource is a Cascading Style Sheet (CSS) file.

### Seletors

**Tags**

- `body` = This gives style to the full body of the page. What you use here applies to all the elements of the page.

  **Properties**
  - `background` = This gives style to the background of an element.

    **Values**
    - `color` = It could be in English, hexadecimal, rgb and others.

## External CSS

It means that the styles are in a independent file with the extension .css.

### CSS Elements

- `<link>` = Is used to link to external resources such as CSS, JavaScript, images, and more. It goes in the head section of the HTML document.

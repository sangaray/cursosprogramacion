# Container

It is a box that contains elements or an element that contains other elements. In general it used a `div` element as a container.

## Properties

- `width` = The width property sets the width of an element.

  **Values**
  - `value px` = The value of the width property is specified in pixels. Is a static value.
  - `value %` = The value of the width property is specified in percentage. Is a relative value. It takes the percentage of the width of the parent element.
  - `auto` = The width is automatically calculated based on the content of the element. It keeps the content within the element and always visible, it does not overflow.

* `margin` = The margin property sets the margin of an element. It can have up to four values: top, right, bottom, and left. The values can be specified in pixels, percentage, or auto. It defines the space outside the element's border.

  **Values**
  - `auto` = It allows to center the element.
